# PROMPT_GENERATION_WORKFLOW.md — Prompt Pack Creation

Purpose:
Standardize how KITT creates high-quality prompt packs.

---

## Step 1 — Research

Use PROMPT_RESEARCH_PROTOCOL.md to identify opportunities.

Output:
Framework
Problem
Opportunity

---

## Step 2 — Validate Demand

Check:

- GitHub stars
- search volume
- developer community size
- tool popularity

---

## Step 3 — Define Pack Type

Choose from:

Skill
Protocol
Pattern
Decision
Persona

---

## Step 4 — Build Pack

Use PROMPT_PACK_TEMPLATE.md.

Include:

- domain knowledge
- rules
- patterns
- workflow
- output format

---

## Step 5 — Stress Test

Ask:

Can another AI read this pack and perform the task correctly?

If not, improve.

---

## Step 6 — Package

Prepare for sale:

- title
- description
- difficulty
- target user
