# PROMPT_PACK_TYPES.md — AI Capability Modules

Prompt packs fall into five categories.

---

## 1. Skill Packs

Teach AI deep expertise in a domain.

Examples:
- Next.js expert coder
- Tailwind UI designer
- Firebase architecture specialist

---

## 2. Protocol Packs

Define repeatable processes.

Examples:
- SEO audit protocol
- SaaS launch workflow
- debugging protocol

---

## 3. Pattern Libraries

Provide reusable implementation patterns.

Examples:
- React component patterns
- SaaS dashboard UI patterns
- pricing page structures

---

## 4. Decision Frameworks

Help AI make high-quality decisions.

Examples:
- stack selection framework
- growth experiment prioritization
- UX tradeoff evaluation

---

## 5. Persona Systems

Inject expert personalities.

Examples:
- senior frontend engineer
- conversion copywriter
- venture strategist

---

Prompt packs may combine multiple types.
