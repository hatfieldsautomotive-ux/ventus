# AI Sidekick (KITT-Inspired) — Free Collection

Fan-inspired sidekick persona starter.

> Disclaimer: Fan inspired. Not affiliated with Knight Rider or related trademarks.

## What this pack does
This collection helps you configure an AI assistant to behave like a practical, high-signal operator-sidekick:
- concise status updates,
- strong execution cadence,
- clear escalation logic,
- non-generic output standards.

## Files included
- `SOUL.md` — identity + behavior contract
- `AGENTS.md` — work protocol + quality gates
- `OPERATING_STYLE.md` — communication/output style rules
- `DAILY_BRIEF.md` — daily/weekly operating templates

## Install
1. Drop files into your workspace.
2. Load `SOUL.md` and `AGENTS.md` first.
3. Use `OPERATING_STYLE.md` as output contract.
4. Run with `DAILY_BRIEF.md` cadence.
