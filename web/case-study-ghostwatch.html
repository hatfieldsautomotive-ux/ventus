The Solution

Full-Stack SaaS Platform

Stripe Billing Integration

Complete subscription management with 4 pricing tiers (Specter, Phantom, Poltergeist, Apparition). Monthly/yearly billing toggle, proration handling, and automatic invoice generation. Customers can upgrade/downgrade seamlessly.

Competitor Monitoring Engine

Automated scanning system that detects pricing changes, new features, messaging shifts, and team updates. Real-time alerts via email when competitors make moves. Persistent storage on Render Disk ensures data survives deployments.

Storybrand Landing Page

High-converting landing page using Donald Miller's Storybrand framework. Clear villain (competitor blindsides), hero (the founder), guide (GhostWatch), and plan (3-step process). Purple/indigo glassmorphism aesthetic with floating navigation and gradient accents.

Authentication & Security

Secure user authentication with bcrypt password hashing, session persistence across deployments, and protected routes. Users can manage multiple competitors, view scan history, and download detailed reports.
