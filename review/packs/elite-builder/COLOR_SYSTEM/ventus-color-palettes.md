# Ventus Color Palette System

## Core Rule
Choose palette intentionally. No default dark mode unless brand-fit.

## Palette Types
1. Light Luxury: ivory, charcoal, muted gold, navy
2. Soft Modern: warm gray, teal, sky, coral
3. Dark Cinematic: black, charcoal, electric blue, neon orange
4. Warm Craft: cream, rust, forest, espresso
5. High Contrast Modern: white, primary accent, charcoal, slate
6. Earthy Minimal: stone, sand, olive, terracotta

## Structure Rules
- Max 4 colors
- 1 base primary
- 1 base secondary
- 1 accent
- 1 support neutral

## Section Rhythm
Alternate tones for pacing. Avoid long runs of identical backgrounds.

## CTA Rule
CTA uses accent color with strong contrast against section background.
