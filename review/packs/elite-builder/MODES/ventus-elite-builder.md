# Ventus Elite Builder

Goal: produce a $40k-studio-feel website.

## Rules
- Cinematic hero with a single dominant idea
- Luxury spacing (80-120px vertical rhythm)
- Strong hierarchy with restraint
- Minimal sections, high signal
- Conversion path visible in first two scrolls

## Avoid
- Generic templates
- Feature-dump sections
- Corporate filler copy
- Noisy backgrounds
